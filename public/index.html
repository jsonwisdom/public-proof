Required fields

  • verification and dispute status
  • counterevidence and rebuttal links
  • correction status and review date

Hard boundaries

  • strong signal is not proof
  • no public record is not proof of absence
  • address association is not control
  • wallet association is not wallet control
  • EAS records a statement; it does not make it true
